Command Name : Patient Information

Description : This commands includes the general information about a patient that is need when doctor patient relationship is established. There are patient particulars that are essential for a doctor to provide medical advice.

Usage:

```
pt 24F wt78kg ht1.8m bed12 unit:ER bg:AB- gcs:e2v2m4
```

if you try to read this, it should be obvious that what does this information contains:

This maps to 
```
A 24 year old female with weight of 78 kg and height of 1.8 meters comes to the Emergency department with a Glassgow comma score of 8 which is severe. Female is known to have blood group AB- and is on bed number 12 in ER.
```
