# Frequently Asked Questions

**Do I need any programming or coding knowledge for this?**
Absolutely not. Think of it like texting another doctor. Just type short acronyms and values, and let the software turn it into a formal note.

**What if I make mistakes or type something the system doesn't know?**
It is completely safe. If you invent a command (like typing `eyes 6/6`), ClinLang will simply group it under "EYES: 6/6" in the final note. It will never break or delete your work.

**Can I write in my own style?**
Yes. You can put commands in any order you want. You can type `vitals` before `hpi`, or `pt wt70 65M`. ClinLang reorganizes everything for you automatically into a standard SOAP format.

**Do I need to type colons for blood tests?**
No. For basic tests, just type them together: `hb9.2`. You only need a colon if you are writing out words, like `cxr:patchy consolidation` or complex titers like `widal:1/60`.

**How does abbreviation expansion work?**
If you type `dm2`, the output will automatically say "Type 2 Diabetes Mellitus". If you type `qds iv`, the output will say "Intravenously, Four times daily". This applies across your entire note to make it look professional.
